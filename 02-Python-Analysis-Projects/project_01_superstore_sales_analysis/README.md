# Superstore Sales Analysis:

## Project Overview:
This project analyses retail sales data to identify patterns in revenue, product demand, customer purchasing behaviour, and regional sales performance. 

The goal is to demonstrate a structured data analysis workflow using Python and pandas.

---

## Dataset:
The dataset used in this project is a retail transaction dataset containing customer, product, regional, and sales information.

**Name:** Superstore Sales Dataset
**Source:** https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting

Dataset Size/Shape:
- Rows: 9800
- Columns: 18

---

## Objective:
The objective of this project is to analyse sales data to understand key business patterns.

The analysis focuses on:
- revenue performance.
- product demand.
- regional sales distribution.
- customer purchasing behaviour.

---

## Workflow:
The analysis follows a structured workflow with each stage documented in a corresponding file.
1. Inspect dataset structure: 'analysis/01_inspect_data.py'.
2. Document findings: 'analysis/02_data_audit.md'.
3. Apply cleaning if necessary: 'analysis/03_cleaning.py'.
4. Perform analysis: 'analysis/04_analyisis.py'.
5. Record insights: 'results/insights.md'.
