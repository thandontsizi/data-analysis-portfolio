# PostgreSQL Fundamentals:
## Overview:
This folder is dedicated to learning the fundamentalsof PostgreSQL, a powerful relational database system. It covers installation, 
navigation, creating databases and tables, inserting data, querying, updating, deleting, and essential commands daily use.


## Objectives:
By the end of this folder, I should be able to:
1. Install and configure PopstgreSQL locally.
2. Navigate the 'psql' terminal efficiently.
3. Create and manage databases and tables.
4. Insert, update, and delete data.
5. Query data using SELECT statements and aggregation.
6. Use PostgreSQL commands and shortcuts for productivity.
7. Apply these fundamentals in mini-projects.

## Structure:
- 'commands.md': A reference for PostgreSQL commands, syntax, and explanations.
- 'tests.md': Simple exercises to practice and verify my knowledge.
- 'projects/': Mini-projects to apply PostgreSQL skills in real scenarios.

## How to Use This Folder:
1. Follow the commands in 'commands.md' to learn PostgreSQL basics.
2. Complete the exercises in 'tests.md' to test your understanding.
3. Apply your knowledge in 'projects/' by creating small databases and running queries.
